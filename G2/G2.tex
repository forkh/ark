\section{Introduktion}
En pipelined MIPS implementation. Disse instruktioner er forventede at virke:
\begin{itemize}
  \item addu, addiu
  \item slt, slti
  \item subu
  \item and, andi
  \item or, ori
  \item lw, sw
  \item beq*
  \item jal*, jr*
\end{itemize}
*Disse instruktioner skal have support for branch delay slot.
